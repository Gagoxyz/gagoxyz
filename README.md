## 🚀 Proyectos destacados

### 🔹 [CesceSync](https://github.com/Gagoxyz/CesceSync)
**Servicio Windows en .NET 8 para sincronizar movimientos de riesgo con la API oficial de CESCE.**  
Stack: .NET 8 (Worker Service), SQL Server, OAuth2  
> Automatiza la sincronización diaria de riesgo, genera alertas, ejecuta SP y mantiene logs persistentes.

---

### 🔹 [CesceSalesSender](https://github.com/Gagoxyz/CesceSalesSender)
**Aplicación CLI en .NET 8 para el envío masivo y paralelo de ventas/facturas a CESCE.**  
Stack: .NET 8, SQL Server, OAuth2, Concurrent Processing  
> Procesa ventas por fechas, las envía a CESCE, registra resultados y mantiene logs con retención.

---

### 🔹 [EnviarCorreusLauncher](https://github.com/Gagoxyz/EnviarCorreusLauncher)
**Utilidad .NET Framework que recibe parámetros desde un externo, obtiene un correo pendiente en SQL Server y genera la petición JSON para ExtraMailOffice.exe.**  
Stack: .NET Framework 4.8, SQL Server, JSON, procesos externos 
> Automatiza el envío de correos corporativos mediante ExtraMailOffice, asegura trazabilidad, gestiona adjuntos y registra el resultado directamente en base de datos.

---

### 🔹 [IronFit](https://github.com/Gagoxyz/ironfit)
**Landing Page responsive y accesible para un gimnasio ficticio**  
Stack: HTML, CSS, JavaScript  
> Centrado en buenas prácticas de maquetación, UX/UI y accesibilidad.

---

### 🔹 [Tic-Tac-Toe con Chat en Tiempo Real](https://github.com/Gagoxyz/tic-tac-toe-socket)
**Juego multijugador con lógica sincronizada y chat por WebSockets**  
Stack: JavaScript, Node.js, WebSocket  
> Partidas en tiempo real, chat integrado y turnos sincronizados.

---

### 🔹 [Todo App estilo Trello](https://github.com/Gagoxyz/todo-app)
**Aplicación fullstack con autenticación, drag & drop y gestión de tareas**  
Stack: Node.js, MongoDB, HTML/CSS/JS  
> Backend propio, rutas protegidas y tablero interactivo.
